Наименование локаций:
Сперва идёт название локации, потом её состояние (FW - First Walkthrough / R - Rest) + добавлять свои правила именования, если понадобиться.

Примеры:
L1_FW
L1 - Location 1,
FW - Первое прохождение

L2_R 
L2 - Локация 2,
R - остальные прохождения (второе, третье, и т.д.)

Наименование уровней:
Сперва название ветки (Голубая, лес, и т.п.), потом какой уровень по счету

Пример:
Blue_1
